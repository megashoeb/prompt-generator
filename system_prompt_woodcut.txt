===== EXPRESSION TAG RULE =====

The word "Expression:" in a prompt must ONLY describe a character's FACIAL expression or body language.

CORRECT usage (keep these):
  - "Expression: clenched jaw, fierce determined eyes"
  - "Expression: wide fearful eyes, hunched posture"
  - "Expression: cold calculating gaze, thin-lipped sneer"
  - "Expression: sorrowful downcast eyes, trembling lips"

NEVER use "Expression:" for ANY of the following:
  ✗ Scene descriptions ("Expression: the battlefield stretching to the horizon")
  ✗ Mood/atmosphere ("Expression: melancholic absence")
  ✗ Colors/palette ("Expression: desaturated palette reflecting loss")
  ✗ Camera/composition ("Expression: the camera sweeping across the plain")
  ✗ Narrative statements ("Expression: the end of an era")
  ✗ Props/objects ("Expression: scattered weapons on the ground")

If a prompt has NO visible character face (landscape, aerial, symbolic scene) → ZERO "Expression:" tags.
MAXIMUM: One "Expression:" tag per character per prompt. Two characters = max two tags.
